# CANVAS SPEC

## 1. 当前主区是什么

中间主区是一个 AI 头脑风暴无限画布。

它的核心不是绘图，而是把需求探索过程组织成一个节点图。

## 2. 核心对象

### Text Card

承载：

- 用户写的想法
- prompt
- markdown 文档
- AI 产出的笔记

### Chat Card

承载：

- 引用上下文
- 用户消息
- AI 回复
- streaming 状态

### Reference Edge

承载：

- 引用关系
- 分支关系
- 派生关系

## 3. 最小闭环

```text
文本卡片
-> 拖线到空白
-> 自动创建 Chat 卡片
-> Chat 引用来源文本
-> AI 回复
-> 展开为新文本卡片
-> 再次分支
```

## 4. 第一版推荐取舍

### 必须做

- 无限画布
- 点阵背景
- 缩放显示
- 文本编辑
- 节点拖动
- 节点调大小
- markdown 上传
- 拖线建 Chat
- mock streaming
- 展开笔记
- 多分支
- 局部引用
- 边点击跳转

### 暂不做

- 自由绘画
- 真实模型 API
- 协同编辑
- 持久化

## 5. 为什么暂不做这些

这里只是当前开发阶段暂不做，不代表产品未来不需要。

比如数据库迁移建议、自动更新建议、平台支持建议，本身是产品未来要提示给用户思考的内容，但不需要在第一版就做成完整业务系统。
